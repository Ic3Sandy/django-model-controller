# Django Common Box Set

Common usage function in a Django project.

## Quick start

1. Install via pip
```
$ pip install django-common-boxset
```

2. Add commons to your INSTALLED_APPS setting like this:
```
INSTALLED_APPS = [
    ...
    'commons',
]
```
